Lesson: Creating a JPA Entity and Spring Data JPA Repository
Junie:
In the directory guru.springframework.springfsintro create a new package called entities. Add a new JPA entity called Greeting. Add an ID property with the type of Integer and create a String property called greeting. Add getters and setters.
In the directory guru.springframework.springfsintro create a new package called repositories. Add a new Spring Data JPA repository for Greeting called GreetingRepository.
Create a @DataJpaTest and test CRUD operations using the GreetingRepository.
